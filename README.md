# Byte Converter

Dieses kleine Projekt ist ein Umrechner für digitale Speichereinheiten. Man kann damit Werte zwischen Bit, Byte, KiB, MiB, GiB, TiB und PiB umrechnen.

## Was macht die App?

- Man gibt eine Zahl ein.
- Man wählt eine Einheit (zum Beispiel MiB).
- Man wählt eine zweite Einheit (zum Beispiel GiB).
- Die App zeigt sofort das Ergebnis.

Es gibt auch einen Knopf, um die beiden Einheiten zu tauschen.

## Verwendete Technologien

- HTML für die Struktur der Seite
- CSS für das Design
- JavaScript für die Berechnung und die Interaktion

Die App läuft komplett im Browser. Man braucht keinen Server und keine Installation.

## Warum dieses Projekt?

Ich habe dieses Projekt erstellt, um meine ersten Programmierkenntnisse zu zeigen. Es ist ein einfaches, aber funktionierendes Beispiel für eine interaktive Webanwendung.
